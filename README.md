# Éclat Store

Éclat Store est une plateforme e-commerce spécialisée dans la vente de produits cosmétiques, pensée d'abord pour le marché malien puis pour une expansion en Afrique de l'Ouest.

## Stack

- Frontend : Next.js, React, TypeScript, TailwindCSS, Framer Motion
- Backend : Node.js, Express, TypeScript
- Base de données : PostgreSQL, Prisma ORM
- Images : Cloudinary

Les paiements en ligne ne sont pas intégrés dans le périmètre actuel.

## Structure

```text
frontend/
backend/
docs/
scripts/
```

## Commandes prévues

```bash
npm install
npm run dev
```

Les commandes nécessitent l'installation des dépendances Node.js.
