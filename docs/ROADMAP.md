# ÉCLAT STORE — ROADMAP DU PROJET

Version : 1.0  
Projet : Éclat Store  
Type : Plateforme e-commerce cosmétique  
Développement : Solo

---

# 1. Introduction

Ce document définit l'évolution prévue d'Éclat Store.

La roadmap permet de :

- organiser les priorités ;
- éviter de développer des fonctionnalités inutiles trop tôt ;
- suivre l'avancement ;
- définir les objectifs à court, moyen et long terme.

Le développement est pensé pour une personne seule.

La priorité est donc :

1. Construire une boutique fonctionnelle.
2. Obtenir les premiers clients.
3. Améliorer progressivement l'expérience.
4. Transformer Éclat Store en plateforme complète.

---

# 2. Vision globale

Objectif final :

Faire d'Éclat Store une référence de la vente de cosmétiques en ligne au Mali puis en Afrique de l'Ouest.

La plateforme doit devenir :

- fiable ;
- rapide ;
- premium ;
- simple à utiliser ;
- adaptée aux habitudes locales.

# 5. Fonctionnalités à éviter au début

Ne pas développer trop tôt :

- marketplace ;
- IA ;
- application mobile ;
- système complexe de fidélité ;
- fonctionnalités sociales.

Ces fonctionnalités demandent une base utilisateur existante.
